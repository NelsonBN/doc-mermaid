# Documentation - Mermaid - Flowcharts
[[Home]](/README.md)



## Basic Structure
A simple flowchart with nodes and connections:

```mermaid
flowchart LR
  A[Start] --> B[Process]
  B --> C[End]
```



## Graph
The syntax also supports to use the `graph` keyword instead of `flowchart`:

```mermaid
graph LR
  A[Start] --> B[Process]
  B --> C[End]
```
